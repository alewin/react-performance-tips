The purpose of this project is to create a community of react developers.
The content of this repository is made up of:
- patterns
- best ptractices
- tools
and useful information learned during during the experience with the React.

# How to contribute?

## 1 Clone the project
```bash
git clone https://github.com/alewin/react-tips
```

## 2 Clone the project
Add a post, within one of the 7 chapters (directories), or refine an existing note, inserting links or other useful tips

```markdown title=chapters
src/docs/{chapter}
```

## 3 Add your name
Since your contribution can be useful to everyone, it is morally right that you insert your github page/twitter nickname 
at the end of the markdown note in the section: "Contributors"

## 4 Pull request
Create a Pull Request (develop)
